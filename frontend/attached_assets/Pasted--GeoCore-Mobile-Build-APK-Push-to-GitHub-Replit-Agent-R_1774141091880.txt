# GeoCore Mobile — Build APK + Push to GitHub
> ابعت لـ Replit Agent بعد ما يخلص بناء الـ React Native app

---

## Step 1 — Verify App is Working

```bash
# Make sure everything compiles
cd mobile
npx expo export --platform android

# Check for TypeScript errors
npx tsc --noEmit
```

Fix any errors before continuing.

---

## Step 2 — Connect to EAS

```bash
# Login to Expo
npx eas-cli login

# Link to project
npx eas-cli init
```

---

## Step 3 — Build APK (Android)

```bash
# Build internal APK for testing
eas build --platform android --profile preview --non-interactive
```

This will:
- Upload code to Expo servers
- Build a real APK file
- Give you a download link

**Save the APK download link.**

---

## Step 4 — Test the APK

Download the APK and install on Android phone:
```
Settings → Security → Allow unknown sources → Install APK
```

Verify these work:
- [ ] App opens without crash
- [ ] Home screen loads with blue header + yellow logo
- [ ] Categories scroll horizontally
- [ ] Listings show with AUCTION/BUY NOW badges
- [ ] Countdown timers are live and counting
- [ ] Bottom nav shows floating yellow + button
- [ ] Login screen works
- [ ] Can browse listings without login

---

## Step 5 — Push to GitHub

```bash
cd mobile

# Make sure .gitignore is correct
cat > .gitignore << 'EOF'
node_modules/
.expo/
dist/
build/
*.jks
*.keystore
.env
.env.local
ios/
android/
EOF

# Push to GitHub under /mobile folder
git init
git add .
git commit -m "feat: React Native mobile app v1.0
- eBay-style UI with Walmart colors (#0071CE + #FFC220)
- Home: Search bar + Categories + Live Auctions + Listings grid
- AUCTION/BUY NOW badges on every card
- Live countdown timers on auctions
- Floating yellow Sell button
- JWT auth with secure token storage
- EAS Build ready (APK + IPA)"

git remote add origin https://github.com/hossam-create/geocore-next.git
git push origin main
```

---
