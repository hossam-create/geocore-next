## Important Rules

1. **Colors:** ONLY use #0071CE (blue) and #FFC220 (yellow) as primary colors
2. **Images:** If no real image, use `https://picsum.photos/400/300?random={id}` as placeholder
3. **Badges:** Every listing card MUST show either 🔨 AUCTION (red) or ⚡ BUY NOW (blue) badge
4. **Countdown:** Every auction card MUST show a live countdown timer
5. **Grid:** Home listings = 2 columns. Search results = list view (like eBay)
6. **Search bar:** Must be in the blue header, prominent and tappable
7. **Sell button:** Middle tab must be a floating yellow circle like eBay's app
8. **DO NOT** change any API calls, data fetching, or business logic
9. **DO NOT** change navigation structure — only the visual design
10. After all changes, test on iOS simulator and confirm it renders correctly
