## TASK-102: Reverse Auction

**What is it:**
Buyer posts what they need and a budget. Sellers compete by offering LOWER prices. Lowest seller wins.

Example: "I need a logo design, budget $200" → Designers bid $150, $120, $90 → $90 wins.

**Changes:**
```go
// Reverse auction: price goes DOWN with each bid
// In PlaceBid:
if auction.AuctionType == "reverse" {
    // Validate: req.Amount < auction.CurrentBid (LOWER is better)
    // seller_id = req.UserID (sellers bid, not buyers)
    // buyer_id = auction.SellerID (the original poster is the buyer)
    minBid = auction.CurrentBid // must be LOWER
    if req.Amount >= minBid {
        response.BadRequest(c, fmt.Sprintf("Reverse auction: bid must be lower than %.2f", minBid))
        return
    }
}
```

**Commit:** `git commit -m "feat(auctions): reverse auction bidding logic"`

// Auto-generate slug after creating listing:
import "regexp"
import "strings"

func generateSlug(title, city string, id uuid.UUID) string {
    // "iPhone 15 Pro Max" → "iphone-15-pro-max"
    slug := strings.ToLower(title)
    re := regexp.MustCompile(`[^a-z0-9]+`)
    slug = re.ReplaceAllString(slug, "-")
    slug = strings.Trim(slug, "-")
    // Add city + short ID to ensure uniqueness
    citySlug := strings.ToLower(re.ReplaceAllString(city, "-"))
    shortID := id.String()[:8]
    return fmt.Sprintf("%s-%s-%s", slug, citySlug, shortID)
}
